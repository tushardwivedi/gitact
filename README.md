# Todo list app
<br>

## cloning project

```
git clone https://github.com/aphrodeosubarno/node-js-todo-app.git
```

## install dependencies

```
yarn install
```

<p>or</p>

```
npm install
```

## run server

```
yarn run dev
```

<p>or</p>

```
npm run dev
```

## open this server at your browser

```
http://localhost:3000
```
